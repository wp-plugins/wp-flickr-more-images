== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the `wp-flickr-more-images` folder to the to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You can see **WP Flickr more images** widget in widgets section.
4. Add it in sidebar and footer and configure as you want.
5. Enjoy!
